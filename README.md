# ColorCollector：身の回りの色を用いた彩色システム

カメラから取得した色情報と物体の動きを利用して、リアルタイムに描画を行うインタラクティブな彩色システムです。

身近な物体から色を取得し、選択した色を使って物体の動きに合わせた軌跡をキャンバス上へ描画します。

彩色後に花火型のフレームを重ねることで、制作中には見えなかった色彩や軌跡が一つの作品として現れます。

完成形を意図的に決めるのではなく、集めた色や動きから偶然生まれる表現を楽しむ制作体験を実現します。

## 特徴

- カメラを使った色の取得
  - 画面左側にリアルタイムのカメラ映像を表示します。
  - カメラ映像をクリックすることで、身の回りにある物の色を描画色として選択できます。

- 物体の動きによる描画
  - 選択した色をカメラ映像から追跡し、その動きに合わせてキャンバス上に描画します。
  - 物体をカメラの前で動かすことで、筆記具を使わずに色を描き重ねる体験を実現しています。

- 花火型フレームによる鑑賞表現
  - 彩色後に、キャンバス上に花火型のフレームを表示します。
  - フレームは時間経過でパターンが切り替わり、完成後の鑑賞にも変化が生まれます。

- 完成形を予測できない制作体験
  - ユーザーは自由に色を選びながら描画しますが、フレームによってどの部分が作品として現れるかは制作中には分かりません。
  - 制作過程で生まれた色や軌跡が、最後に予想していなかった形で現れる体験を楽しめます。

- 直感的な彩色体験
  - クリックで色を選び、物体を動かして描画し、キー操作でフレームを表示できます。
  - 複雑な設定を必要とせず、身の回りの色を集めながら制作できます。

## 体験の流れ

1. プログラムを起動すると、左側にカメラ映像、右側に白いキャンバスが表示されます。

2. カメラ映像内の使いたい色をクリックします。

3. 選択した色の物体をカメラの前で動かすと、その動きに合わせてキャンバスが塗られます。

4. 別の場所をクリックすると選択色が更新され、複数の色を用いて自由に彩色できます。

5. `C` キーを押すと花火型のフレームが表示され、
   塗り重ねた色が花火の形として切り取られます。

6. フレームは時間経過によって切り替わり、完成作品の見え方が変化します。

## 操作方法

| 操作 | 内容 |
| --- | --- |
| カメラ映像をクリック | 描画に使う色を選択 |
| 物体を動かす | キャンバスに描画 |
| `C` キー | 花火フレームの表示 |


## コンセプト

本アプリケーションは、ひっかき絵にある「完成形が予想できない制作過程のワクワク感」から着想を得ています。

ひっかき絵では、キャンバス表面を削ることで下に隠れた色が現れます。制作中は、最終的にどのような色や形が見えるのかを完全には予測できません。

ColorCollectorでは、その不確実性を「カメラを通して身の回りの色を集め、キャンバスに塗り重ねる」体験を通じて表現しました。彩色している段階では完成形を予測できませんが、花火型のフレームを重ねた瞬間に、これまで集めた色彩が一つの作品として現れます。
