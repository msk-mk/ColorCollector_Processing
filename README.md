# ColorCollector：身の回りの色を用いた彩色システム

## 概要

本アプリケーションは「ひっかき絵」の「完成形が予想できない制作過程のワクワク感」から着想を得ている。
ひっかき絵では、キャンバス表面を削ることで下に隠れた色が徐々に現れ、最終的にどのような色や絵になるのか分からないまま制作が進行する。
そのため、完成後の見え方を意図して絵を描くことができないという特徴を持つ。
一方で本アプリケーションでは、その「完成形が予想できない」という体験を共通としつつも、異なるアプローチをとっている。
ユーザーはカメラを通して身の回りの物を使い、キャンバスに色を自由に塗り重ねていく。
しかし、その時点では最終的にどのような作品になるのかを予測することができず、完成後の見え方を意図して塗り進めることができない体験となっている。
そして彩色後、フレームによってキャンバスが切り取られることで、初めてそれまでに塗られた色が一つの絵の中で活かされるようになる。
その結果、塗り重ねている色がどのような形で活かされるのか期待しながら描き進めるという、制作過程そのもののプロセスにワクワク感が生まれる体験となっている。

## 体験の概要

カメラを通して現実空間の色を取得し、それらを自由に塗り重ねながら作品を制作する。
しかし、制作中は最終的にどのような見え方になるのかを把握することができず、完成形を予測できないまま彩色が進行する。
そして最後に花火型のフレームが重ねられることで、それまで塗り重ねてきた色が絵の中で活かされる。
これにより、完成形への期待感を持ちながら彩色を行い、また、予想していなかった見え方が現れる驚きを楽しんで鑑賞することができる。

## 体験フロー

### 1. プログラム起動

アプリを起動すると、画面左側にカメラ映像が表示され、リアルタイムで周囲の環境が入力される。
画面右側には真っ白なキャンバスが表示される。
ここに彩色を行う。

### 2. 色の選択と彩色

カメラ映像内の特定の箇所をクリックすることで、その部分の色が取得され、選択される。
選択した色に対応する物体をカメラ内で動かすことで、その動きに応じてキャンバスが塗りつぶされていく。
また、別の箇所をクリックすることで選択色を更新し、異なる色による彩色を自由に行うことができる。

### 3. 彩色の終了

彩色を終えたタイミングでキーボード入力を行うことで、彩色フェーズが終了する。

### 4. フレームの表示と鑑賞体験

彩色後、花火の形にくり抜かれたフレームがキャンバス上に重ねて表示される。
このフレームによって、それまでに塗られた色が花火の形として切り抜かれる。
その結果、自由に塗り重ねた色が予想していなかったカラフルな花火の絵として活かされ、完成形を初めて目にする驚きとワクワク感を伴う鑑賞体験となる。
